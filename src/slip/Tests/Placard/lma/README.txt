Liste des programmes (cours SINF1160)
-------------------------------------
adr.lma : 
  sous-programme de calcul du déterminant avec
  passage des paramètres par adresse

badSyntax.lma :
  cas d'erreur quand le fichier ne se termine pas
  par une ligne vide...

moyenne.lma :
  calcul d'une moyenne en virgule flottante

prog.lma : calcul du déterminant (incorrect) 
  (erreur : opérations en complément à deux au lieu de flottants)

prog1.lma : idem sans erreurs 
  (+ instructions pour mettre une chaîne dans 3 registres.)

prog3.lma : autre version de la moyenne

val.lma : 
  sous-programme de calcul du déterminant avec
  passage des paramètres par valeur


------------------------------------
Programmes d'examen (cours SINF1160)
------------------------------------

containsXX.lma : 
  sous-programme pour tester la présence de deux occurrences
  successives d'une même valeur dans un tableau.

factorielle.lma :
  calcul de la factorielle

int2float.lma :
  conversion d'une valeur entière (complément à deux)
  en virgule flottante (en Java : int --> float)

renverser.lma :
  renverser une chaîne de caractères
